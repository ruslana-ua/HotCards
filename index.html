Hotcards is an online application allowing content creators & site editors to quickly generate customizable cards.
The cards can be integrated on web pages and shared on social networks and by email.

An automated Schema Markup code is automatically created when a card is generated. The code improves the way search engines read the card information. This allows them to facilitate the process of associate authors with their contents.
